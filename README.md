
# 💸 StudentBudget - Grupo 4

Presupuesto para estudiantes.




## ⚠️ Problema que resuelve tu pagina Web

Muchos estudiantes no administran correctamente el dinero que reciben para transporte, alimentación y gastos personales, lo que provoca que gasten más de lo que tienen disponible.



## ✅ Propuesta de Valor y Publico Objetivo

“StudentBudget” permite registrar ingresos y gastos de forma rápida 
y sencilla, mostrando automáticamente el saldo disponible para ayudar a los estudiantes a administrar mejor su dinero. Principalmente enfocada en estudiantes de bachillerato, estudiantes universitarios, jóvenes que desean controlar sus gastos personales.
